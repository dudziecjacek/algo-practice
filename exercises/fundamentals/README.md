# Fundamentals

Not tied to one specific assessment task, but drills the general skills
the briefing lists (hash sets, boundary reasoning, O(n)). A classic
Codility warm-up.

| Exercise | What it drills |
|----------|----------------|
| [01-smallest-missing-positive](01-smallest-missing-positive) | `Set` lookup, pigeonhole bound `1..N+1`, edge cases, guaranteed return |
| [02-contains-duplicate](02-contains-duplicate) | `Set` early-exit, O(n) vs the O(n²) pair check |
| [03-array-diff](03-array-diff) | `Set` from `b` once, filter `a` — avoiding the `.includes` nested loop |

Topics: hash sets · indexing · edge cases · time complexity · reasoning
about *why* a bound is provably complete.
